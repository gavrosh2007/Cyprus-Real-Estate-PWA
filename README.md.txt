# Real Estate PWA

Прогрессивное веб-приложение для размещения объявлений о продаже недвижимости.

## Возможности

- Добавление объектов с фото (drag & drop или выбор файлов)
- Просмотр детальной информации с фотографией
- Фильтрация по владельцам (кнопка «Список»)
- Переключение языка (русский/английский)
- Установка на устройство как PWA
- Работа офлайн
